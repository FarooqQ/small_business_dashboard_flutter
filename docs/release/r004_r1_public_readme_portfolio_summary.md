## 1. Project Overview

Small Business Dashboard Starter is a Flutter desktop dashboard starter package designed for small businesses.

It helps business owners and managers understand sales, profit, orders, average order value, low stock items, category performance, and report previews using prepared CSV data.

The project is designed as a reusable freelance-ready dashboard package that can be customized for different small business clients.

---

## 2. What Problem It Solves

Many small businesses keep their data in Excel or CSV files, but they do not have a clean dashboard to understand business performance quickly.

This project helps convert structured CSV data into a visual dashboard that shows useful business indicators.

It is useful for:

- small shops
- local stores
- small warehouses
- product-based businesses
- early-stage business owners
- freelance dashboard demos
- portfolio presentation

## 5. Data Sources

The project currently uses two CSV files:

### sales.csv

Used for:

- sales
- profit
- orders
- average order
- top product
- sales by category
- categories

### inventory.csv

Used for:

- low stock count
- low stock items

The project is Excel-compatible because clients can prepare data in Excel and export it as CSV.
