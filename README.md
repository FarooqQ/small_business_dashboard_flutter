# Small Business Dashboard Flutter

A Flutter-based small business dashboard demo built as a service-ready portfolio project.

This project demonstrates how Flutter and Dart can be used to build a lightweight business dashboard foundation with clean structure, report preparation, export content logic, unit testing, documentation, and release readiness.

The project is published as a public portfolio repository to demonstrate practical software engineering skills and the ability to prepare a small business application for client-facing delivery.

---

## Project Overview

Small Business Dashboard Flutter is a lightweight dashboard project designed for small business use cases.

It is not a large ERP system. Instead, it provides a clean and practical foundation that can be extended into a custom dashboard solution for clients who need simple business monitoring tools.

The project focuses on:

- clean Flutter project structure
- dashboard-oriented development
- business report preparation
- text and CSV export content generation
- unit testing for core logic
- documentation-driven development
- release preparation
- public portfolio presentation

---

## Project Purpose

The main purpose of this project is to show how a small business dashboard can be developed in a structured and maintainable way.

Many small businesses need simple tools to understand their sales, inventory, reports, and performance indicators without depending on complex or expensive ERP systems.

This project can be used as a starting point for building:

- sales dashboards
- inventory dashboards
- admin dashboards
- report modules
- lightweight ERP modules
- client-specific business dashboards

## How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/FarooqQ/small_business_dashboard_flutter.git
```

### 2. Enter the Project Folder

```bash
cd small_business_dashboard_flutter
```

### 3. Install Dependencies

```bash
flutter pub get
```

### 4. Run the Project

```bash
flutter run
```

For Windows desktop:

```bash
flutter run -d windows
```
